# Infinite Upgrade Card / 无限升级卡

[English](#english) | [中文](#中文)

<a name=""english""></a>
## English

A Minecraft 1.12.2 mod that adds infinite upgrades and infinite factory installer.

### Features

#### Items

1. **Infinite Upgrade**
   - Right-click on a machine while sneaking to install speed and energy upgrades
   - Default: 8 speed upgrades + 8 energy upgrades
   - Configurable via config file

2. **Super Infinite Upgrade**
   - Right-click on a machine while sneaking to install more speed and energy upgrades
   - Default: 17 speed upgrades + 32 energy upgrades
   - Configurable via config file

3. **Infinite Factory Installer**
   - Right-click on a factory to upgrade it to Ultimate tier
   - Works with MekCEu and MoreMachine factories
   - No item consumption in any game mode

### Requirements

- Minecraft 1.12.2
- Forge 14.23.5.2864 or higher
- Mekanism CE Unofficial (required)
- MekanismCeU-MoreMachine (optional, for factory tier upgrades)

### Configuration

The mod generates a configuration file at config/infinite_upgrade_card.cfg where you can customize:

- Infinite Upgrade speed count (default: 8)
- Infinite Upgrade energy count (default: 8)
- Super Infinite Upgrade speed count (default: 17)
- Super Infinite Upgrade energy count (default: 32)

### Building

```bash
./gradlew build
```

The compiled jar will be in uild/libs/infinite_upgrade_card-1.0.jar

### Installation

1. Install Minecraft Forge for 1.12.2
2. Install Mekanism CE Unofficial
3. Place the jar file in the mods folder

## 中文

一个Minecraft 1.12.2模组，添加了无限升级和无限工厂安装器。

### 特性

#### 物品

1. **无限升级**
   - 潜行状态下右键机器安装速度和能量升级
   - 默认：8个速度升级 + 8个能量升级
   - 可通过配置文件自定义

2. **超高级无限升级**
   - 潜行状态下右键机器安装更多速度和能量升级
   - 默认：17个速度升级 + 32个能量升级
   - 可通过配置文件自定义

3. **无限工厂安装器**
   - 右键工厂将其升级到终极等级
   - 支持MekCEu和MoreMachine工厂
   - 任何游戏模式下都不会消耗物品

### 需求

- Minecraft 1.12.2
- Forge 14.23.5.2864 或更高版本
- Mekanism CE Unofficial（必需）
- MekanismCeU-MoreMachine（可选，用于工厂等级升级）

### 配置

模组会在 config/infinite_upgrade_card.cfg 生成配置文件，你可以自定义：

- 无限升级速度数量（默认：8）
- 无限升级能量数量（默认：8）
- 超无限升级速度数量（默认：17）
- 超无限升级能量数量（默认：32）

### 构建

```bash
./gradlew build
```

编译后的jar文件位于 uild/libs/infinite_upgrade_card-1.0.jar

### 安装

1. 安装Minecraft Forge 1.12.2版本
2. 安装Mekanism CE Unofficial
3. 将jar文件放入 mods 文件夹
